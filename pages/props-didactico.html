🎯 Props en React

Descubre cómo hacer componentes inteligentes y reutilizables desde cero

🚀¿Por qué Props son el Superpoder de React?

Imagina que pudieras crear una máquina que produce tarjetas de presentación. En lugar de hacer una máquina diferente para cada persona, creas UNA sola máquina que puede personalizar cada tarjeta según los datos que le des.

🏭 Analogía de la Fábrica de Tarjetas

Sin Props: Tienes que construir una máquina diferente para cada persona

Con Props: Una sola máquina que recibe diferentes "materiales" y produce tarjetas únicas

"Los Props son como los ingredientes de una receta: cambias los ingredientes y obtienes un plato diferente, pero la receta sigue siendo la misma."

1Nivel Principiante: ¿Qué son las Props?

📚Definición simple de Props

Props = Properties = Propiedades

Las props son datos que le pasas a un componente desde fuera, como si fueran parámetros de una función.

En funciones normales:

function saludar(nombre) {
  return "Hola " + nombre;
}

saludar("María"); // "Hola María"

En componentes React:

function Tarjeta({ nombre }) {
  return <h1>Hola {nombre}</h1>;
}

<Tarjeta nombre="María" />
🔍Comparación con HTML

¡Ya conoces las props! En HTML, los atributos son como props:

HTML con atributos:

<img src="foto.jpg" 
     alt="Mi foto" 
     width="200" />

Los atributos src, alt, width personalizan la imagen

React con props:

<Tarjeta titulo="Mi Blog" 
         color="azul" 
         tamaño="grande" />

Las props titulo, color, tamaño personalizan el componente

🎨Tu primer componente con props

Paso a paso - Componente Saludo

1. Componente sin props (limitado):
function Saludo() {
  return <h1>¡Hola Juan!</h1>;
}

// Solo puede saludar a Juan 😢
2. Componente con props (flexible):
function Saludo({ nombre }) {
  return <h1>¡Hola {nombre}!</h1>;
}

// Ahora puede saludar a cualquiera 🎉
<Saludo nombre="María" />    // ¡Hola María!
<Saludo nombre="Carlos" />   // ¡Hola Carlos!
<Saludo nombre="Ana" />      // ¡Hola Ana!

🔑 Concepto clave: Un componente + diferentes props = resultados diferentes

🎭 Analogía del Actor

El componente es como un actor que puede interpretar diferentes personajes. Las props son el guión que le dice qué personaje interpretar en cada escena.

🔧Props con valores por defecto

¿Qué pasa si no pasas todas las props?

❌ Sin valores por defecto:
function Boton({ texto, color }) {
  return <button style={{ color }}>{texto}</button>;
}

<Boton texto="Click" /> // color será undefined 😱
✅ Con valores por defecto:
function Boton({ texto, color = "blue" }) {
  return <button style={{ color }}>{texto}</button>;
}

<Boton texto="Click" /> // color será "blue" 🎉

Método alternativo con OR (||):

function Tarjeta({ titulo, color }) {
  const colorFinal = color || "#f0f0f0"; // Si no hay color, usa gris claro
  
  return (
    <div style={{ backgroundColor: colorFinal }}>
      <h3>{titulo}</h3>
    </div>
  );
}
🎯Props para controlar comportamiento

Las props no solo son para mostrar texto

1. Props booleanas (verdadero/falso):
function Mensaje({ texto, esImportante }) {
  return (
    <p style={{ 
      color: esImportante ? "red" : "black",
      fontWeight: esImportante ? "bold" : "normal"
    }}>
      {texto}
    </p>
  );
}

<Mensaje texto="Mensaje normal" />
<Mensaje texto="¡Urgente!" esImportante={true} />
2. Props para números:
function BarraProgreso({ porcentaje }) {
  return (
    <div style={{ width: "200px", backgroundColor: "#eee", borderRadius: "10px" }}>
      <div style={{ 
        width: `${porcentaje}%`, 
        height: "20px", 
        backgroundColor: "#4CAF50",
        borderRadius: "10px"
      }}></div>
    </div>
  );
}

<BarraProgreso porcentaje={75} /> // Barra al 75%

3Nivel Avanzado: Props Opcionales y Condicionales

🖼️Props opcionales (imagen opcional)

Componente que muestra imagen solo si existe

function TarjetaCompleta({ titulo, descripcion, imagen, color }) {
  return (
    <div style={{ 
      backgroundColor: color || "white", 
      padding: "1rem", 
      borderRadius: "8px",
      textAlign: "center"
    }}>
      {imagen && <img src={imagen} alt={titulo} style={{ width: "100px", height: "100px" }} />}
      <h3>{titulo}</h3>
      <p>{descripcion}</p>
    </div>
  );
}

Usando el componente:

// Tarjeta CON imagen
<TarjetaCompleta 
  titulo="React"
  descripcion="Librería de JavaScript"
  imagen="react-logo.png"
  color="#e1f5fe"
/>

// Tarjeta SIN imagen
<TarjetaCompleta 
  titulo="HTML"
  descripcion="Lenguaje de marcado"
  color="#fff3e0"
/>

🔍 Cómo funciona: {imagen && <img.../>} significa "si imagen existe, muestra la etiqueta img"

🎨Props para estilos dinámicos completos

Máximo control sobre la apariencia

function TarjetaPersonalizable({ 
  titulo, 
  descripcion, 
  colorFondo, 
  colorTexto, 
  tamaño,
  sombra,
  borde 
}) {
  const estilos = {
    backgroundColor: colorFondo || "white",
    color: colorTexto || "black",
    padding: tamaño === "grande" ? "2rem" : "1rem",
    fontSize: tamaño === "grande" ? "1.2rem" : "1rem",
    boxShadow: sombra ? "0 4px 8px rgba(0,0,0,0.1)" : "none",
    border: borde || "none",
    borderRadius: "8px",
    margin: "1rem"
  };

  return (
    <div style={estilos}>
      <h3>{titulo}</h3>
      <p>{descripcion}</p>
    </div>
  );
}

Ejemplos de uso:

// Tarjeta básica
<TarjetaPersonalizable 
  titulo="Básica" 
  descripcion="Tarjeta simple" 
/>

// Tarjeta con todo personalizado
<TarjetaPersonalizable 
  titulo="Premium"
  descripcion="Tarjeta con todos los extras"
  colorFondo="#f0f8ff"
  colorTexto="#1565c0"
  tamaño="grande"
  sombra={true}
  borde="2px solid #2196f3"
/>
🔄Props como funciones (eventos)

Pasando funciones como props

function BotonInteligente({ texto, alHacerClick, color }) {
  return (
    <button 
      onClick={alHacerClick}
      style={{ 
        backgroundColor: color || "#4CAF50", 
        color: "white",
        padding: "0.5rem 1rem",
        border: "none",
        borderRadius: "4px",
        cursor: "pointer"
      }}
    >
      {texto}
    </button>
  );
}

// En el componente padre:
function App() {
  const saludar = () => alert("¡Hola!");
  const despedir = () => alert("¡Adiós!");

  return (
    <div>
      <BotonInteligente texto="Saludar" alHacerClick={saludar} color="green" />
      <BotonInteligente texto="Despedir" alHacerClick={despedir} color="red" />
    </div>
  );
}

4Nivel Experto: Props con Estructuras Complejas

📊Props con Arrays (listas)

Pasando listas como props

function ListaCompras({ elementos, titulo }) {
  return (
    <div style={{ border: "1px solid #ccc", padding: "1rem", borderRadius: "8px" }}>
      <h3>{titulo}</h3>
      <ul>
        {elementos.map((elemento, index) => (
          <li key={index}>{elemento}</li>
        ))}
      </ul>
    </div>
  );
}

// Usando el componente:
const compras = ["Leche", "Pan", "Huevos", "Frutas"];

<ListaCompras 
  titulo="Mi Lista de Compras" 
  elementos={compras} 
/>

💪Ejercicio: Crea tu propio componente Carta

Objetivo: Crear un componente que muestre información de una persona

📋 Requisitos del componente:
  • ✅ Debe recibir: nombre, apellido, edad, profesión, foto (opcional)
  • ✅ Debe mostrar la foto solo si existe
  • ✅ Debe tener un color de fondo personalizable
  • ✅ Si la edad es mayor a 65, debe mostrar "Jubilado" en lugar de la profesión
🎯 Plantilla para empezar:

  nombre, 
  apellido, 
  edad, 
  profesion, 
  foto, 
  colorFondo 
}) {
  // Tu código aquí...
  
  return (
    <div style={{ backgroundColor: colorFondo || "white" }}>
      // Tu JSX aquí...
    </div>
  );
}
✨ Solución:
Ver solución (intenta primero) 
function Carta({ nombre, apellido, edad, profesion, foto, colorFondo }) {
  const mostrarProfesion = edad > 65 ? "Jubilado" : profesion;
  
  return (
    <div style={{ 
      backgroundColor: colorFondo || "#f5f5f5",
      padding: "1.5rem",
      borderRadius: "10px",
      textAlign: "center",
      border: "1px solid #ddd",
      margin: "1rem"
    }}>
      {foto && <img src={foto} alt={nombre} style={{ 
        width: "100px", 
        height: "100px", 
        borderRadius: "50%",
        marginBottom: "1rem"
      }} />}
      <h2>{nombre} {apellido}</h2>
      <p>{edad} años</p>
      <p style={{ fontStyle: "italic" }}>{mostrarProfesion}</p>
    </div>
  );
}

// Ejemplo de uso:
<Carta 
  nombre="María"
  apellido="González"
  edad={30}
  profesion="Desarrolladora"
  foto="maria.jpg"
  colorFondo="#e3f2fd"
/>
🎮Desafío: Componente interactivo con props

Crea un componente Contador que:

  • Reciba un valor inicial como prop
  • Reciba un color para el botón como prop
  • Reciba una función que se ejecute al hacer click
function Contador({ valorInicial, colorBoton, alCambiar }) {
  // Pista: necesitarás useState para el estado interno
  const [count, setCount] = useState(valorInicial || 0);
  
  const incrementar = () => {
    const nuevoValor = count + 1;
    setCount(nuevoValor);
    alCambiar && alCambiar(nuevoValor); // Llama a la función si existe
  };

  return (
    <div style={{ textAlign: "center", padding: "1rem" }}>
      <h2>Contador: {count}</h2>
      <button 
        onClick={incrementar}
        style={{ 
          backgroundColor: colorBoton || "blue",
          color: "white",
          padding: "0.5rem 1rem",
          border: "none",
          borderRadius: "4px"
        }}
      >
        Incrementar
      </button>
    </div>
  );
}

🎓Resumen: Lo que has aprendido

🧠 Conceptos clave que dominas:

  • Props básicas: Pasar texto, números y booleanos
  • Props de estilo: Personalizar colores y apariencia
  • Props opcionales: Usar renderizado condicional
  • Props por defecto: Evitar errores con valores predeterminados
  • Props complejas: Arrays, objetos y funciones
"Las props son el lenguaje que usan los componentes para comunicarse. Dominando las props, puedes crear componentes que se adapten a cualquier situación."
