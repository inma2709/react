¿Qué es un Layout?

Un Layout es la estructura base que se repite en todas las páginas de una aplicación web. En React, se utiliza para envolver el contenido que cambia (las rutas o páginas) con elementos que permanecen fijos, como el header, la barra lateral o el footer.

De este modo, todas las páginas heredan automáticamente la misma disposición, lo que hace el desarrollo más rápido, ordenado y accesible.

Ventaja: El layout permite mantener la coherencia visual y funcional en toda la app.

📐 Estructura básica de un Layout

  • Header: Cabecera con el título o logotipo del sitio.
  • Nav: Menú de navegación principal.
  • Sidebar: Barra lateral opcional (filtros, enlaces extra, etc.).
  • Footer: Pie de página con créditos o enlaces.
  • Outlet: Espacio dinámico donde se “montan” las páginas.

⚙️ Cómo se configura

Para usar un layout en React con rutas:

  1. Importa Outlet desde react-router-dom.
  2. Crea un componente Layout.jsx que incluya tus zonas fijas.
  3. En App.jsx, declara un <Route element=<Layout />> que envuelva las páginas hijas.
            
            import { Outlet } from "react-router-dom";
            export default function Layout() {
              return (
                <>
                  <Header />
                  <Nav />
                  <Sidebar />
                  <main>
                    <Outlet />
                  </main>
                  <Footer />
                
              );
            }

De esta manera, todas las rutas que dependan de ese Layout compartirán su diseño general.
