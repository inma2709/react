📌 ¿Para quién es este curso?

Si ya sabes crear páginas web con HTML, CSS y JavaScript básico, este curso te ayudará a dar el siguiente paso hacia el desarrollo moderno.

No necesitas ser un experto en JavaScript. Si puedes crear funciones, manejar eventos del DOM y entiendes conceptos como variables y arrays, ¡ya tienes todo lo necesario!

Aprenderemos React conectando cada concepto nuevo con lo que ya conoces, para que la transición sea natural y sin estrés.

🎯 Lo que lograrás al final

  • Entender por qué React existe: Verás las limitaciones del JavaScript puro y cómo React las soluciona.
  • Pensar en componentes: Aprenderás a dividir tu página en piezas reutilizables (como los divs, pero más poderosos).
  • Manejar datos dinámicos: Harás que tu página cambie automáticamente cuando los datos cambien.
  • Crear tu primera aplicación: Una página interactiva que realmente funciona.
  • Publicarla online: Para que puedas mostrarla y sentirte orgulloso/a de tu progreso.

🎯 Tu Ruta de Aprendizaje: Desarrolla en React

🚀 Antes de empezar

Asegúrate de tener Node.js instalado en tu computadora. Puedes descargarlo desde nodejs.org (versión LTS recomendada).

📦 Día 1 - "¡Hola React!"

SETUP

Configuramos todo y creamos nuestro primer componente. No continúes hasta que veas tu app corriendo en el navegador.

🎯 Lograrás: Crear proyecto React, entender la estructura de archivos, escribir tu primer componente.

🧩 Día 2 - "HTML que piensa"

JSX

Descubrimos JSX - escribir HTML dentro de JavaScript. Entiende el código, no solo lo copies. .

🎯 Lograrás: Dominar JSX, crear componentes reutilizables, usar props básicas.

🚀 Día 3 - "Tu Portafolio Personal"

PROYECTO

Construimos un portafolio real usando todo lo aprendido. Tu primer proyecto que puedes mostrar.

🎯 Lograrás: Estructurar una app completa, usar props avanzadas, manejar arrays de datos.

⚡ Día 4 - "Páginas que reaccionan"

INTERACTIVO

Añadimos interactividad: botones, formularios y estado. Experimenta y rompe cosas.

🎯 Lograrás: Manejar eventos, cambiar contenido dinámicamente, crear formularios controlados.

🏁 Día 5 - "Tu primera app completa"

FINAL

Integramos todo en una tarjeta digital interactiva. Demuestra todo lo aprendido.

🎯 Lograrás: Combinar todos los conceptos, crear algo único, publicar tu proyecto online.

💡 Consejos para el éxito

  • Entiende todo el código no lo memorices simplemente entiende la lógica detrás de cada línea.
  • No avances hasta que no entiendas lo de detrás.Intentamos interiorizar conocimientos.
  • Experimenta e investiga. Sólo probando vas a aprender.

💡 ¿Por qué React?

😓 JavaScript puro es complicado:

  • Buscar elementos en el DOM manualmente
  • Actualizar cada parte cuando algo cambia
  • Manejar eventos y estado por separado

😎 Con React es más fácil:

Describes cómo quieres que se vea tu página y React se encarga del resto.

¡Es como tener un asistente que actualiza todo automáticamente!

🔄 De lo que sabes a React

React no es tan diferente de lo que ya conoces. Aquí te muestro las conexiones:

  • HTML → JSX: Sigues escribiendo etiquetas, solo que ahora tienen más poder.
  • CSS → Estilos en React: Tus conocimientos de CSS se aplican igual, solo cambia dónde los escribes.
  • JavaScript → Componentes: En lugar de funciones que manipulan el DOM, creas funciones que devuelven HTML.
  • Variables → Estado: Como las variables de JS, pero cuando cambian, la página se actualiza automáticamente.

¡Verás que React hace que todo sea más organizado y fácil de mantener!
